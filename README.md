# Tiny Wins Journal

A minimal daily journal to log small wins, track streaks, and visualize your consistency over time.

Live app: https://cherish22k.github.io/Tiny-Wins-Journal/

## Features
- Log one win per day with a category tag
- Streak counter to track consecutive days
- GitHub-style heatmap showing your activity
- All data saved locally in your browser

## Built with
HTML, CSS, JavaScript — no frameworks, no installs.
